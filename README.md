# Enden — 一些方便的程式系統

一套以 Python 撰寫的輕量命令列工具集，提供以下功能：

| 工具 | 說明 |
|------|------|
| 待辦事項管理器 | 新增、列出、完成與刪除任務，資料自動存檔 |
| 計算機 | 支援 `+` `-` `*` `/` `//` `%` `**` 的運算式計算 |
| 單位換算器 | 長度、重量、溫度三大類別換算 |
| 計時器 | 倒數計時與碼錶功能 |

## 需求

- Python 3.8+（無需第三方套件）

## 使用方法

```bash
python main.py
```

啟動後會顯示互動式主選單，依提示輸入對應數字即可進入各功能。

## 執行測試

```bash
python tests.py -v
```
